🧪 Instructions de Test

  1. Lancez le jeu : ./start.sh puis http://localhost:8007
  2. Observez le terrain : Décors colorés en haut et en bas
  3. Testez les collisions : Touchez le terrain → perte de vie
  4. Vérifiez les ennemis : Apparaissent dans la zone centrale
  5. Chronométrez les tirs : Ennemis tirent toutes les 5 secondes
  6. Console (F12) : Vérifiez les logs de terrain et collisions
